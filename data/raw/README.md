## 分类检索规则

Agent 在回答用户问题时，应：
1. **分析问题领域**：判断用户问题属于哪个或哪些法律类别
2. **选择检索范围**：优先在相关法律类别中检索
3. **跨类检索**：如问题涉及多个领域（如劳动+合同），可同时检索多个类别
4. **兜底检索**：如无法确定类别，或专项检索结果不足，再进行全库检索

## 新增法律类别

新增法律文件时，请：
1. 在 `data/raw/` 下创建对应 `.md` 文件
2. 在本索引表中添加一行记录
3. 重新构建向量库以纳入新法律
