# 背景音乐目录

## 使用说明

将你想要作为背景音乐的 MP3 文件放入此目录，系统会自动识别。

## 命名规则

音乐按数字命名，播放列表会按文件名排序：
- `1.mp3`
- `2.mp3`
- `3.mp3`
- ...
- `9.mp3`

最少 1 首，最多 9 首。

## 支持格式

- ✅ MP3（推荐，兼容性最好）
- ✅ WAV
- ✅ OGG

## 推荐规格

- **比特率**：128kbps ~ 320kbps
- **时长**：建议 3~10 分钟（循环播放）
- **风格**：轻音乐、纯音乐、Lo-Fi、自然音效等
- **大小**：建议每首控制在 10MB 以内

## 功能说明

放松模式支持：
- 播放 / 暂停控制
- 音量调节（0% ~ 100%）
- 循环模式：单曲循环 / 列表循环 / 顺序播放
- 自动切换下一首
