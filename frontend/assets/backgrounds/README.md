# 背景图片目录

## 使用说明

将你想要作为背景的图片放入此目录，支持以下格式：
- JPG / JPEG
- PNG
- WebP
- GIF（动态图片）

## 命名规则

图片按数字命名，系统会自动按顺序轮播：
- `1.jpg`
- `2.jpg`
- `3.jpg`
- ...
- `9.jpg`

最少 1 张，最多 9 张。

## 关于动态图片和视频

### GIF 动态图片
✅ **支持**
- 将 `.gif` 文件放入此目录即可
- 注意：GIF 文件通常较大，建议控制在 5MB 以内
- 虚化效果对 GIF 同样生效

### 视频背景
⚠️ **有限支持**
- Streamlit 原生不直接支持视频背景
- 如需视频背景，需要自定义 HTML + CSS 实现
- 建议使用 MP4 格式，通过 `<video>` 标签注入
- 视频背景会占用较多资源，可能影响页面性能

## 推荐图片规格

- **尺寸**：1920×1080 或更高（适配大多数屏幕）
- **比例**：16:9 宽屏
- **大小**：每张建议 500KB ~ 2MB
- **风格**：建议选择低对比度、偏暗的图片，避免影响文字可读性

## 虚化效果说明

开启虚化后，背景会应用 `backdrop-filter: blur(10px)` 效果：
- 让文字更清晰可读
- 营造层次感
- 不影响前景内容的交互
